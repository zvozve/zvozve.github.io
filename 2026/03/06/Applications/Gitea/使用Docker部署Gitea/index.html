🚨 注意事项

  1. 版本一致性：确保新旧服务器 Gitea 版本相同

  5. 定时任务：如果有备份定时任务需要重新配置

  • 新服务器部署相同版本的 Gitea
  • 恢复备份数据
  • 更新配置文件中的域名/IP
  • 启动新服务器服务
  • 测试所有功能正常
  • 更新 DNS 或客户端配置

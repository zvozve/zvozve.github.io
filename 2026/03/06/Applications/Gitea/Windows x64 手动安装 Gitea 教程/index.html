📋 系统要求

  • Windows 7/8/10/11 或 Windows Server 2012+
  • 64位操作系统
  • 建议 2GB+ 内存
  • 1GB+ 可用磁盘空间

📥 下载 Gitea

方法 1：从官网下载

  1. 访问 Gitea 下载页面
  2. 选择最新版本
  3. 下载 gitea-[version]-windows-amd64.exe

🔧 首次运行配置

  1. 访问 Gitea：

    • 打开浏览器访问 http://localhost:3000
    • 如果是远程访问，使用 http://服务器IP:3000

  2. 初始设置：

    • 数据库设置（如果使用内置 SQLite 则已自动配置）
    • 管理员账户设置
    • 服务器域名和基础 URL
    • 邮件服务器配置（可选）

  3. 保存配置：

    • 点击 “立即安装”
    • 系统会自动重启并应用配置
